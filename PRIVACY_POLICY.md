# Privacy Policy

Effective date: 2026-07-04

## Overview

FFXIVKR_GLAMOUR_TRANSLATOR displays Korean item and dye names on EORZEACOLLECTION and MIRAPRI pages by querying item/search pages on `ff14.tar.to`.

## Data collection

This extension does not collect, store, sell, or share personal information.

## Data processed by the extension

The extension reads item and dye names displayed on supported glamour pages in order to show Korean translation badges.

To retrieve Korean item/dye names, the extension sends the item or dye name as a search keyword to `https://ff14.tar.to/`. These requests are used only to display the translation result in the current browser page.

## Local storage

This extension does not save personal information to local storage or remote servers.

## Third-party services

This extension queries `ff14.tar.to` to retrieve item and dye search results. The third-party site may receive the searched item or dye keyword as part of the request URL.

## Limited Use disclosure

The extension uses data only to provide or improve the visible item/dye translation feature. It does not use data for advertising, profiling, resale, or unrelated purposes.
